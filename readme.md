# **iDo 微信小程序介绍文档（简版）**

## 项目背景

App Store等应用商店上线的专注考研信息类app寥寥无几（非留学向），而最具代表性的考研帮app虽然功能强大，但并未推出与其功能性相当的小程序。大学生每天接收着大量的信息，微信等社交软件更是其接收信息的重要来源之一。获取考研信息的需求虽不如背单词或查字典那样频繁，但也时刻影响着大学生的努力方向和学习策略。

## 项目理念

就像每天早起需要看天气预报来决定一天的穿着与出行，我们推出的这款小程序也让用户每天翻阅目标专业与目标院校的信息以及相关竞赛的推荐信息，从而决定短期与长期的努力方向与竞赛准备。

## 使用说明

- 首次登录后，可在院校信息界面浏览全部院校的专业信息及招生信息等信息，此时若浏览目标信息界面，将会提示转到个人信息界面进行目标设置。

- 院校信息界面显示的院校信息按院校研究生指导能力排名。

- 设置目标信息后，目标信息界面上方院校信息与专业信息及相应招生信息等信息默认显示第一志愿内容，可点击“切换”更换成其他志愿。
