# **iDo 微信小程序介绍文档（简版）**

## 项目背景

App Store等应用商店上线的专注考研信息类app寥寥无几（非留学向），而最具代表性的考研帮app虽然功能强大，但并未推出与其功能性相当的小程序。大学生每天接收着大量的信息，微信等社交软件更是其接收信息的重要来源之一。获取考研信息的需求虽不如背单词或查字典那样频繁，但也时刻影响着大学生的努力方向和学习策略。

## 项目理念

就像每天早起需要看天气预报来决定一天的穿着与出行，我们推出的这款小程序也让用户每天翻阅目标专业与目标院校的信息以及相关竞赛的推荐信息，从而决定短期与长期的努力方向与竞赛准备。

## 使用说明

### 新用户登陆

- 首次登录后，可在院校信息界面浏览全部院校的专业信息及招生信息等信息，若检测到到点击动作，将会引导跳转到个人信息界面进行个人考研目标设置。

- 院校信息界面显示院校校徽，排序按照院校研究生指导能力标准进行。

- 设置目标信息后，引导用户点击目标信息。

- 目标信息界面直接显示用户的第一志愿院校校徽以及院校的名称，排名，以及第一志愿专业名称，可以添加校训。下方显示四个模块，分别对应该院校该专业的招生信息，招生简章，复试调剂，相关竞赛，初步确定显示三年内的信息。

### 老用户使用

- 登陆后直接进入目标信息板块，交由用户使用

## 功能描述

1. 提供一部分的院校的排名，标准为院校研究生指导能力。

2. 提供设定个人目标功能，支持单用户设定三个院校，三个专业方向，系统作优先显示和特别提示。

3. 提供修改志愿的功能

4. 提供查看自己的志愿的院校该专业的招生信息，招生简章，复试调剂，相关竞赛的功能。

5. 在目标信息，和个人信息页每日刷新一句鼓励格言。
